# 카이스트 스터디 트랙커 (KAIST Study Tracker)

Made by 박종모, 신재영, 최연우 of GDSC KAIST

Inspired by 열품타

## Getting started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
- this app works in expo go app(android, ios) or web browser

## Stack

- React Native
- Expo
- Firebase
